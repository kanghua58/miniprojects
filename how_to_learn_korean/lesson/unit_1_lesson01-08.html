Unit 1: Lesson 1

n this lesson, you will learn the basic structure in Korean sentences and how to create sentences using the word 이다 (to be). Using 이다, you will learn how to create sentences like “that man is a teacher” and “this thing is a pen.” In addition, some essential grammar notes will be presented to get you started correctly in your path to learn Korean!
